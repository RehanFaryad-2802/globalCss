Responsive Behavior

🔄 Automatic Adjustments

The system automatically adjusts columns for different screen sizes:

Breakpoint Behavior
> 1000px All column classes display as specified
800-1000px col-9 and col-1 switch to horizontal scroll
600-900px col-7 and col-8 switch to horizontal scroll
450-800px col-3 becomes 2 columns, col-4 becomes 3 columns
< 600px col-2 becomes 1 column, col-3/4 become 2 columns
< 450px All columns become 1 column (full width)
Note: Add .nowrap to any column container to prevent responsive adjustments
