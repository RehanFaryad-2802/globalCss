Spacing Utilities

Comprehensive margin and padding classes with pixel and rem values, including directional and axis-specific controls

Margin Classes

↔ Basic Margin

Apply margin to all sides of an element

m[value][unit]
Examples: m10 = margin: 10px m2r = margin: 2rem

↕ Directional Margin

Control margin on specific sides

m[t|r|b|l][value][unit]
Side Codes: t = top, r = right, b = bottom, l = left mt15 = margin-top: 15px mbr10 = margin-bottom: 10rem

↔ X-Axis Margin

Margin on left and right sides simultaneously

mx[value][unit]
Examples: mx10 = margin-left: 10px; margin-right: 10px mx3r = margin-left: 3rem; margin-right: 3rem

↕ Y-Axis Margin

Margin on top and bottom sides simultaneously

my[value][unit]
Examples: my20 = margin-top: 20px; margin-bottom: 20px my5r = margin-top: 5rem; margin-bottom: 5rem

Padding Classes

↔ Basic Padding

Apply padding to all sides of an element

p[value][unit]
Examples: p20 = padding: 20px p3r = padding: 3rem

↕ Directional Padding

Control padding on specific sides

p[t|r|b|l][value][unit]
Side Codes: t = top, r = right, b = bottom, l = left pt5 = padding-top: 5px pl10r = padding-left: 10rem

↔ X-Axis Padding

Padding on left and right sides simultaneously

px[value][unit]
Examples: px15 = padding-left: 15px; padding-right: 15px px4r = padding-left: 4rem; padding-right: 4rem

↕ Y-Axis Padding

Padding on top and bottom sides simultaneously

py[value][unit]
Examples: py10 = padding-top: 10px; padding-bottom: 10px py6r = padding-top: 6rem; padding-bottom: 6rem

Allowed Values

🔢 Pixel Values

Default unit (no suffix needed)

Value Example
5, 10, 15, 20, 30 m10, p15
Directional: t/r/b/l mt5, pr10
X/Y Axis: x/y mx20, py15

📏 REM Values

Add "r" suffix for rem units

Value Example
1, 2, 3, 5, 10, 30 m2r, p5r
Extended: 4, 6, 15, 20 mx4r, py6r
